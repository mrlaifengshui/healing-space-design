## 圖片規格建議

- **格式**: JPG (壓縮後) 或 WebP (更好的壓縮率)
- **質量**: 高質量但經過優化，文件大小控制在 200-500KB
- **色調**: 以綠色、米色、木質色調為主，營造自然療癒氛圍
- **風格**: 現代簡約、自然親生命、專業高端

## 臨時解決方案

如果暫時沒有圖片資源，可以使用以下方式：

1. **Unsplash** 免費高質量圖片
   - 搜索關鍵字: meditation room, zen space, healing space, mindfulness
   - 網址: https://unsplash.com/

2. **Pexels** 免費圖片庫
   - 搜索關鍵字: meditation, wellness, interior design, zen
   - 網址: https://www.pexels.com/

3. **佔位圖片服務** (開發測試用)
   ```html
   <!-- 在 HTML 中暫時使用 -->
   <img src="https://placehold.co/800x600/2c5f2d/white?text=Healing+Space" alt="療癒空間">
   ```

## 圖片優化建議

使用以下工具優化圖片：

1. **TinyPNG** - https://tinypng.com/
   - PNG/JPG 壓縮工具

2. **Squoosh** - https://squoosh.app/
   - Google 開發的圖片優化工具

3. **ImageOptim** (Mac)
   - 本地圖片優化軟體
