## 🌟 專案簡介

本網站是一個專業的療癒空間設計服務平台，專注於提供：

- **科學化靈性空間設計服務**：結合傳統風水智慧與現代科學方法
- **量子身心靈空間淨化儀**：專業產品銷售與諮詢
- **全方位設計解決方案**：涵蓋個人家居、企業空間、療癒中心

## 📋 核心特色

### 設計理念三大支柱

1. **親生命設計 (Biophilic Design)**
   - 將自然元素融入室內空間
   - 透過綠植、自然光、天然材質重建人與自然的連結

2. **感官療癒設計 (Sensory Design)**
   - 視覺、聽覺、嗅覺、觸覺的整合設計
   - 創造多層次的療癒體驗

3. **量子風水能量場 (Quantum Energy Field)**
   - 科學儀器檢測空間能量場
   - 數據化分析並優化空間佈局

## 🎯 目標市場

- **香港市場**：小坪數家居淨修室設計
- **台灣市場**：個人家庭、企業、高端療癒中心

  - B2C: 高端個人住宅業主

## 🚀 快速開始

### 本地開發

1. **克隆專案**
```bash
git clone https://github.com/mrlaifengshui/healing-space-design.git
cd healing-space-design
```

2. **啟動本地伺服器**
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

3. **瀏覽網站**
```
開啟瀏覽器訪問: http://localhost:8000
```

### 部署到 GitHub Pages

1. **推送到 GitHub**
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

2. **啟用 GitHub Pages**
   - 進入專案設置 (Settings)
   - 找到 Pages 選項
   - Source 選擇 main 分支
   - 保存後即可訪問

3. **訪問網站**
```
https://mrlaifengshui.github.io/healing-space-design
```
