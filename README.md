## ✨ 核心特色

### 🌿 三大設計支柱

- **自然空間設計** - 運用自然元素、光線和空氣流動，營造如同置身大自然的療癒環境
- **靈修功能設計** - 通過能量場分析與優化，提升空間的靈性頻率，支持冥想與靜修
- **感官療癒設計** - 優化視覺、聽覺、嗅覺、觸覺體驗，為身心帶來全方位的療癒感受

### 🎯 針對性解決方案

| 方案 | 描述 | 適用場景 |
| :--- | :--- | :--- |
| **個人療癒空間** | 為臥室、書房或家庭靜修室打造私人療癒聖地 | 個人家居 |
| **家居靈修室** | 專為冥想、瑜伽和靈性修行設計的獨立靜修空間 | 家庭靜修 |
| **企業 Wellness Room** | 為企業打造員工身心健康的靜修空間，提升工作效率 | 企業辦公 |
| **Spa 療癒中心** | 高端療癒中心的完整靈性空間設計方案 | 商業療癒 |

### 🔬 量子療癒空間淨化儀

一個三合一的空間淨化和能量優化解決方案：

- **改善空間粒子** - 空氣淨化過濾懸浮粒子及細菌病毒，釋放森林級數的負離子
- **提升能量意識** - 量子調頻驅走負能量及提升正能量
- **優化空間氣場** - 分析及擺設方位提升空間氣場運轉，轉運效應立竿見影

## 🚀 快速開始

### 前置要求

- Node.js 18.0+
- pnpm 或 npm

### 安裝

```bash
# 克隆項目
git clone https://github.com/yourusername/healing-space-design.git
cd healing-space-design

# 安裝依賴
pnpm install

# 啟動開發伺服器
pnpm dev
```

訪問 `http://localhost:3000` 查看網站。

### 構建

```bash
# 生成生產版本
pnpm build

# 預覽生產版本
pnpm preview
```

## 🎨 設計系統

### 色彩主題

我們採用了療癒綠、寧靜藍和科技紫的三色組合，營造既療癒又專業的視覺體驗：

- **療癒綠** (`oklch(0.55 0.15 142.5)`) - 代表自然、生機和信任
- **寧靜藍** (`oklch(0.50 0.14 200)`) - 代表寧靜、專業和科技
- **科技紫** (`oklch(0.50 0.16 270)`) - 代表靈性、創新和高端

### 排版

- **標題字體** - 現代無襯線字體，傳達專業感
- **正文字體** - 易讀的無襯線字體，確保長文本的可讀性
- **行高** - 1.6-1.8，優化閱讀體驗

### 動畫和互動

所有動畫都經過精心設計，速度適中（0.3-0.6 秒），不會分散訪客注意力：

- **淡入動畫** - 頁面和區段優雅淡入
- **滑入動畫** - 內容依序進入，營造流暢感
- **卡片懸停效果** - 卡片在滑鼠懸停時輕微提升並增強陰影
- **脈衝動畫** - CTA 按鈕持續微妙脈衝，吸引注意力
